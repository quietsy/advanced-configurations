## Requirements

The [online draw.io](https://app.diagrams.net/) or the [self-hosted draw.io](https://github.com/jgraph/docker-drawio), here is an example compose:
```Yaml
  drawio:
    image: jgraph/drawio:alpine
    container_name: drawio
    ports:
        - 8080:8080
    restart: always
```
